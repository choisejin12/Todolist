# 📌 ToDoList 웹 서비스

## 📝 프로젝트 소개
할 일을 기록하고 관리할 수 있는 웹 기반 ToDoList 서비스입니다.  
사용자가 일정을 추가하고 상태를 관리할 수 있도록 구현하여  
일상적인 업무 및 일정 관리를 효율적으로 돕는 것을 목표로 했습니다. :contentReference[oaicite:0]{index=0}

---

## 🚀 주요 기능

### 🔐 회원가입 / 로그인
- 이메일 기반 회원가입 및 로그인 기능 구현
- 인증된 사용자만 서비스 이용 가능

### 📋 할 일 관리 (CRUD)
- 할 일 추가 / 수정 / 삭제 기능 구현
- 직관적인 UI를 통해 빠른 일정 관리 가능

### 🔄 상태 관리
- 할 일 상태를 **미완료 / 진행중 / 완료**로 구분
- 라디오 버튼을 활용하여 상태 변경 가능

### 🖱️ 인터랙션 UI
- hover 시 수정 / 삭제 버튼 표시
- 사용자 경험(UX)을 고려한 UI 설계
